Bootstrap defaults

font-size is 14px and line-height is 1.428 applied to body and paragraphs

Emphasis

Class lead will give larger font size, lighter weight and a taller line height.

<p class="lead"> ... </p>

Small tag set text at 85% the size of the parent

<small>some text</small>

Strong tag is rendered as bold text

<strong>some text</strong>

Em tag is rendered as italic text

<em>some text</em>

Inline text elements

Marked text - For highlighting a run of text due to its relevance in another context, use the <mark>tag.

You can use the mark tag to <mark>highlight</mark> text.

Deleted text - For indicating blocks of text that have been deleted use the <del> tag.

<del>This line of text is meant to be treated as deleted text.</del>

Strikethrough text - For indicating blocks of text that are no longer relevant use the <s>tag.

<s>This line of text is meant to be treated as no longer accurate.</s>

Inserted text - For indicating additions to the document use the <ins> tag.

<ins>This line of text is meant to be treated as an addition to the document.</ins>

Underlined text - To underline text use the <u> tag.

<u>This line of text will render as underlined.</u>

Media Queries

Extra small devices (xs - less than 768px)

No media query, since this is the default in Bootstrap

Small devices (sm - 768px and up)

@media (min-width: @screen-sm-min)
{...}

Medium devices (md - 992px and up)

@media (min-width: @screen-md-min)
{...}

Large devices (lg -1200px and up)

@media (min-width: @screen-lg-min)
{...}

Class .visible-xs-*

Visible at xs but hidden at sm, md and lg

Class .hidden-xs-*

Hidden at xs but visible at sm, md and lg

Responsive Column Resets

At certain breakpoints, your columns don't clear quite right as one is taller than the other. To fix that, use a combination of a .clearfix and our responsive utility classes.

<div class="clearfix visible-xs-block"/div>

Offseting Columns

Move columns to the right using .col-md-offset-* classes. These classes increase the left margin of a column by * columns. For example, .col-md-offset-4 moves .col-md-4 over four columns.

<class="col-md-4 col-md-offset-4">.col-md-4 .col-md-offset-4 </div>

Column ordering

Easily change the order of our built-in grid columns with .col-md-push-* and .col-md-pull-* modifier classes.

<class="col-md-9 col-md-push-3">.col-md-9 .col-md-push-3 </div>
<class="col-md-3 col-md-pull-9">.col-md-3 .col-md-pull-9 </div>

Nesting columns

To nest your content with the default grid, add a new .row and set of .col-sm-* columns within an existing .col-sm-* column. Nested rows should include a set of columns that add up to 12 or fewer (it is not required that you use all 12 available columns).

Supported Form Controls

Bootstrap supports the following form controls: input, textarea, checkbox, radio and select

Most common form control types are:text, password, datetime, datetime-local, date, month, time, week, number, email, url, search, tel and color.

Inputs will be only fully styled if their type is properly declared.

<input type"text" class="form-control" placeholder="Text Input">
