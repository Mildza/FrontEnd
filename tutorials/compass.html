Compass

Compass needs ruby to be installed, then start gem and install compass

gem install compass

To create new project and install compass

compass create new_project

To add compass to existing directory, navigate to parrent directory and run

compass install compass

In header we need to add some lines

<link href="/stylesheets/screen.css" media="screen, projection"
 rel="stylesheet" type="text/css" />
<link href="/stylesheets/print.css" media="print"
rel="stylesheet" type="text/css" />
<!--[if IE]>
<link href="/stylesheets/ie.css" media="screen, projection"
rel="stylesheet" type="text/css" />
<![endif]-->

.scss files are stored in sass folder and .css files are in stylesheets folder

We need to change path to css folder

<link rel="stylesheet" type="text/css" href="stylesheets/style.css">

To compile your .scss to .css run

compass watch

In .scss file we must to add

@import 'compass';
